-- Pasos para ejecutar correctamente el proyecto ShopNext --


__________ Únicamente para localhost __________

........... Prerequisitos
1. Tener instalado XAMPP
2. Base de datos actualizada


______ Pasos
1. Descargar el proyecto correspondiente.
2. Redirigir la carpeta a la carpeta de htdocs (C:/xampp/htdocs o parecido)
3. Ejecutar XAMPP
4. Encender los botones de Apache y MySQL
5. Presionar el botón "Admin" de Apache
   Esto de dejará en una dirección similar a http://localhost/dashboard/
6. Agregar el nombre de la carpeta (en este caso main)

RECORDAR, PARA QUE EJECUTE CORRECTAMENTE TIENE QUE ESTAR LA BASE DE DATOS DE SHOPNEXS EN MySQL, PARA ESTO:

1. La base de datos está en la carpeta "database". Importar en MySQL.
2. Para eso, crear la base de datos con el mismo nombre "shopnext"
3. Entrar a "Importar" / "Seleccionar archivo" / "Elejir el archivo en .sql" / "Importar"
